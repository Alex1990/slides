一条 CSS 规则语法

Component value：其中构成之一是带有单位的数值，称之为维度（dimensions），比如前面的24px。

CSS 单位类别

  1. 长度单位
  2. 角度单位
  3. 时间单位
  4. 频率单位
  5. 分辨率单位

长度单位

  • 绝对长度（Absolute lengths）：比如px

  • 相对长度（Rela

    • 字体相对长度（font-relative lengths）：比如em、rem
    • 视口百分比长度（viewport-percentage lengths）：比如vw、vh

function linkify( selector ) {
  if( supports3DTransforms ) {

    var nodes = document.querySelectorAll( selector );

    for( var i = 0, len = nodes.length; i < len; i++ ) {
      var node = nodes[i];

      if( !node.className ) {
        node.className += ' roll';
      }
    }
  }
}
